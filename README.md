# StretchBreak ⟳

Лёгкое приложение для Windows, которое напоминает делать перерывы на разминку — и **не мешает во время игры**.

Живёт в системном трее, по заданному интервалу присылает напоминание «пора размяться». Во время полноэкранной игры экран **не перекрывается** — вместо этого приходят обычные уведомления.

## Возможности

- ⏰ Напоминание о перерыве через заданный интервал (по умолчанию — каждые 60 минут)
- 🔔 **Мягкий режим (по умолчанию):** уведомление Windows без звука; чтобы закрыть — нужно кликнуть по нему, иначе оно повторяется каждые 10 секунд
- 🟣 **Жёсткий режим (опция):** резкое полноэкранное перекрытие с обратным отсчётом на всех мониторах
- 🎮 **Не мешает играм:** во время полноэкранной игры (в т.ч. «окно без рамки») перекрытия не будет — только уведомления. Игры определяются автоматически по размеру окна + по списку процессов
- 🖥️ Корректная работа с несколькими мониторами
- ⚙️ Окно настроек: интервал, длительность, список игр, автозапуск с Windows, звук, пауза
- 🪟 Сворачивается в трей при закрытии, как настоящее приложение

## Установка

1. Скачай `StretchBreak.exe` из раздела [**Releases**](../../releases).
2. Положи в любую папку и запусти двойным кликом.
3. Появится окно настроек и иконка в трее. Готово — установка и зависимости не нужны.

> **SmartScreen / антивирус.** Приложение не подписано цифровой подписью, поэтому Windows может показать предупреждение: «Подробнее» → «Выполнить в любом случае». Если файл пришёл заблокированным: ПКМ → Свойства → «Разблокировать».

## Использование

- **Двойной клик по иконке в трее** — открыть настройки.
- **Крестик окна** — свернуть в трей (приложение продолжает работать).
- **Выход** — через меню трея (ПКМ по иконке → Выход).
- **«Проверить сейчас»** — сразу показать перерыв для проверки.

## Настройки

| Параметр | Описание |
|---|---|
| Интервал между перерывами | Как часто напоминать (минуты) |
| Длительность перерыва | Длина заглушки в жёстком режиме (секунды) |
| Повтор уведомления | Через сколько секунд повторять уведомление, если не закрыто |
| Жёсткий режим | Резкое перекрытие экрана вместо уведомления (не рекомендуется при играх) |
| Игры | Процессы, при которых показывать только уведомления |
| Запуск с Windows | Автозапуск при входе в систему |

Настройки хранятся в `settings.json` рядом с программой.

## Запуск из исходников

Зависимости не нужны — используется встроенный Windows PowerShell.

```powershell
powershell.exe -STA -NoProfile -ExecutionPolicy Bypass -File StretchBreak.ps1
```

Или тихий запуск без окна консоли — двойной клик по `StretchBreak.vbs`.

> Файл `StretchBreak.ps1` должен быть в кодировке **UTF-8 with BOM**, иначе под Windows PowerShell 5.1 ломается кириллица.

## Сборка .exe

```powershell
Install-Module ps2exe -Scope CurrentUser
Invoke-ps2exe -inputFile StretchBreak.ps1 -outputFile StretchBreak.exe -noConsole -STA -iconFile StretchBreak.ico -title StretchBreak
```
